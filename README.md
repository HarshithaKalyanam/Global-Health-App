 Disease Category Prediction System

A Machine Learning web application built using Streamlit and Scikit-learn to predict disease categories based on patient details.

 Features

* Predicts disease category using Machine Learning
* User-friendly Streamlit web interface
* Dropdown-based patient input system
* TF-IDF Vectorization for text processing
* Logistic Regression classification model
* Live deployment using Streamlit Cloud

 Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-learn
* TF-IDF Vectorizer
* Logistic Regression

 Project Structure

```text
Global-Health-App/
│
├── app.py
├── data.csv
├── requirements.txt
└── README.md
```


 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/HarshithaKalyanam/Global-Health-App.git
```

2. Navigate to project folder

```bash
cd Global-Health-App
```

3. Install required libraries

```bash
pip install -r requirements.txt
```

4. Run Streamlit app

```bash
streamlit run app.py
```
